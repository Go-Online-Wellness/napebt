Documentation Basics

The new corporated theme comes as a refresh of the existing layout on pages and also brings a simple yet powerful set of styles which can be easily reused. It is based on the UIKIT framework so it comes with all the powerful features of such library. Below you'll find an overview of the custom components made for the theme and an example on how to implement them.

General info

  • All code must be added inside a section, as the CSS provides custom styling made for sections. A section can be a logical part of the page referring to an specific topic or sub-topic.
  • It is recommended to append the first section of a page with a .section-padding-top in order to provide certaing padding from the .principal-hero class (more on that below).

Principal hero

The principal hero is the top image at the beginning of a page (such as the one you can see at the top of the page), it has to be the second element of a page, right after the Nunita Sans font import. To set the principal hero and an image, you should follow the following syntax. 

<div class="principal-hero" style="background-image=url(url_to_image)"></div>

Typography

Font Family

The theme uses the Nunita Sans font, so for each page the following import must be added at the beginning of the file 

<!-- Start Nunito  Sans Font-->
<link href="https://fonts.googleapis.com/css?family=Nunito+Sans:300,400,600,700,800,900&amp;display=swap" rel="stylesheet" />
<!-- End Nunito Sans Font -->

Font Semantics

You can use up to 5 different font style variations according to the HTML semantics, along with several different font sizes, in order to use them, you'll need to type the corresponding tag along with their corresponding class

Lists

Use .item-list on any list to provide custom styling depending on the context (such as a card or accordion).

  • This is item list one
  • This is item list two
  • This is item list three
<ul class="item-list">
  <li>This is item list one</li>
  <li>This is item list two</li>
  <li>This is item list three</li>
</ul>

Buttons

There are two kinds of button .general-button and .minimal-button, the only difference between them is their size. They also come in three different colors



<button class="general-button uk-button dark-blue">General dark blue</button>
<button class="general-button uk-button light-blue">General light blue</button>
<button class="general-button uk-button white">General White</button>

<button class="minimal-button uk-button dark-blue">Minimal dark blue</button>
<button class="minimal-button uk-button light-blue">Minimal blue</button>
<button class="minimal-button uk-button white">Minimal White</button>

The same styling and types of button can also be applied to links
